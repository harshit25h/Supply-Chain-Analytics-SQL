# Supply Chain Analytics using SQL

## Objective

Analyze inventory, demand, and warehouse performance using SQL.

## Tools Used

- SQL
- SQLite

## Dataset

The project contains:
- Products Table
- Inventory Table
- Demand Table
- Warehouse Table

## Key SQL Concepts Used

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- INNER JOIN
- LEFT JOIN
- Subqueries
- CASE WHEN

## Business Questions Solved

1. Which warehouse has the highest inventory?
2. Which products are understocked?
3. Which warehouses have inventory greater than 100?
4. Which products have no inventory records?
5. Which warehouse stores the most products?

## Key Insights

- Delhi warehouse has the highest inventory.
- Phone was identified as an understocked product.
- LEFT JOIN helped identify products without inventory records.
- Warehouse inventory utilization was analyzed using aggregate functions.
